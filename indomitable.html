Project Reflection :

This game was the first group project I had worked on, as well as the first analogue game I ever produced. There were definitely some unique challenges this project made me privy to such as coordination between several people producing assets, general project management, and balancing a game in such a way where it is challenging yet engaging. We also had to put consideration into the ease at which a user would be able to understand the rules and how to play.
